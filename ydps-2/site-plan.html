Content

Log-in

The Login page allows users who already signed up to enter their email and password to access the website. It checks if the details match and gives a message based on the result.

Images for the Page 1

Sign Up

The Sign-Up page lets new users create an account by entering their name, email, and password. It checks if the information is complete and then saves the account so the user can log in later.

Images for the Page 2

Wireframes

Log-in

The Log-in page allows users to enter their email and password to access their account. JavaScript will be used to validate the input fields, provide instant feedback for errors, and securely handle the login process by checking credentials before allowing access.

Sign Up

Asks for the first name, last name, mobile number, email, and password. JavaScript will be used to validate each input field, provide instant feedback for errors, and securely handle the sign-up process by checking for completeness and proper formatting before creating the account.
